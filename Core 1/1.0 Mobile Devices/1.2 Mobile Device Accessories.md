## Stylus
![](<./assets/1.2 Mobile Device Accessories/file-20260220214722667.png>)[^1]
- **Digital stylus**
	- A more advanced writing tool
	- A precise input method
	- Usually connected via Bluetooth, therefore wireless
- **The stylus communicates directly to the device**
	- Pressure sensitivity, programmable buttons, etc. 
	
- **Must be compatible with the tablet**
	- e.g., *Apple iPad uses an Apple Pencil*
	- Make sure the stylus is compatible with the device you're connecting to.
	- Devices with their own compatible stylus usually create the best possible connection and enable most of the features for the end user.

## Headsets
![](<./assets/1.2 Mobile Device Accessories/file-20260220204135132.png>)[^2]
- **Hands-free audio**
	- Earphones and a microphone
![](<./assets/1.2 Mobile Device Accessories/file-20260220202124814.png>)[^3]
- **Wired**
	- USB connections are common on laptops 
	- Connects to 3.5 mm **TRRS** (tip-ring-ring-sleeve) connector
	- Analog audio jack
	- Older iPhone models usually use lightning port for connecting external audio devices
![](<./assets/1.2 Mobile Device Accessories/file-20260220203958866.png>)[^4]
- **Wireless**
	- Bluetooth headsets
	- Do not need the TRRS connector
	- Runs on battery (can run out)

## Speakers
![](<./assets/1.2 Mobile Device Accessories/file-20260220215557353.png>)[^5]
- Speakers on the phone don't provide much fidelity.
- **Mobile audio**
	- Battery-powered and portable speaker system
- **Wireless connection**
	- Usually connected to your mobile device wirelessly via Bluetooth
	- Allows users to take these speakers anywhere within the range of mobile device.
- **Stereo sound**
	- Small package

## Camera/webcam
![](<./assets/1.2 Mobile Device Accessories/file-20260220215741251.png>)[^6]
- **Internal or external**
	- Commonly external on desktops, internal on laptops/ tablets/ smartphones
- **Accessible from multimedia applications, photo utilities, and video conferencing.**
	- Real-time video communication

## Docking station
![](<./assets/1.2 Mobile Device Accessories/file-20260220205513326.png>)[^7]
- **Docking station**
	- A quick way to connect and disconnect a laptop without having to touch any of the cables that are on the desk (whether in the office or at home). 
		- Use an external keyboard and mouse
		- Extend existing laptop interfaces
		- Add additional functionality
		- Desktop adapter cards
		- Avoid cable issues

## Port replicator
![](<./assets/1.2 Mobile Device Accessories/file-20260220215140179.png>)[^8]
- If you don't need support for additional adapter cards and want something more portable, you might want to use a port replicator.
- **Similar to a docking station**
	- Does not commonly have an expansion card option
	- Usually connects using USB
	- Can work with almost any laptop or desktop computer.
	- Can also include a lot of different connectors:
		- USB connections and audio
		- built-in MicroSD and SD card readers
		- Additional USB and video output options

## Trackpad
![](<./assets/1.2 Mobile Device Accessories/file-20260220215857371.png>)[^9]
- **Replaces the external mouse**
	- Useful in tight working areas
	- Very common on laptops
- **External options**
	- Battery powered
	- Bluetooth or wired connection
- **Drag and tap**
	- Or use multiple-finger input for right-clicking, zooming, and window control
- **Enable and disable**
	- Avoid inadvertent mouse clicks and movements
	- Many laptops come with a feature to disable the trackpad using a function key.

## Drawing pad
![](<./assets/1.2 Mobile Device Accessories/file-20260220220026363.png>)[^10]
- **Uses an active stylus with an external digitizer**
	- Gives the same precise input on a desktop/ laptop computer as a tablet device.
- **Usually a third-party device that connects to your computer over Bluetooth or a USB connection.**
	- Supported across many operating systems
